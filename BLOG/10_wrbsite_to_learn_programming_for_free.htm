5 Website to learn programming for free :

  There are so many resources to learn to program for free without losing any money. So in our time, there is nothing to waste or lose to learn something new or just search for a piece of information. 1- W3schools

it's a wonderful website that can help you learn any programming language for free, like javascript and python... based on practice ,and after finish each chapter you can exercise what you have received and take a quiz.

it's a website for learning to program for free it's cover the most used programming language from A to Z. and the interesting thing about this website it gives you certification after complete each course, so you can add it to your portfolio and make it like an experience in your professional life.

3-Corsera

Codecademy is an interactive online platform that offers free learning of the basics of twelve programming languages, such as Python, PHP, JavaScript, Java, C ++
